# O que é TypeScript?

TypeScript é uma linguagem baseada em JavaScript que adiciona **tipagem estática** ao código.

Isso significa que podemos informar:
- que tipo de dado uma variável deve armazenar
- que tipo de valor uma função deve receber e retornar
- qual é a estrutura de um objeto

## Por que TypeScript existe?

Em JavaScript, muitos erros só aparecem quando o programa está sendo executado.
Em TypeScript, esses erros podem ser identificados **antes da execução**, durante o desenvolvimento.

## Importante

- TypeScript **não substitui** o JavaScript
- Todo código TypeScript vira JavaScript
- O navegador **não executa TypeScript diretamente**
