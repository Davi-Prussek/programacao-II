# Programação II

## Olá!

Este repositório é um projeto pessoal e independente, criado com o objetivo de
estudar e praticar a linguagem TypeScript.

Apesar de não fazer parte oficialmente da trilha do Curso Técnico em Informática
do IFC Araquari, este material foi pensado como uma continuação natural dos
conceitos estudados em Programação I.

#

# Atenção

Para um melhor aproveitamento deste repositório, é recomendado que o aluno:

- Tenha concluído ou esteja cursando **Programação I**, com foco em lógica de
  programação usando JavaScript  
   Material do professor Fábio Longo de Moura disponível
  [aqui](https://github.com/ldmfabio/Programacao)

- Possua uma base sólida em:
  - variáveis e operadores
  - estruturas de decisão
  - estruturas de repetição
  - arrays
  - funções
  - objetos

> Este repositório **não substitui** o material oficial do curso, mas serve como
> introdução e aprofundamento no uso do TypeScript.

#

## Objetivo da Programação II

Este repositório tem como objetivo o estudo da linguagem TypeScript, abordando
seus principais conceitos e sua utilização prática.

O conteúdo parte dos conhecimentos de lógica de programação já adquiridos,
permitindo ao aluno aprender a usar o TypeScript de forma estruturada.

#

## Organização do Repositório

O conteúdo está organizado de forma progressiva, partindo da introdução ao
TypeScript até a prática dos principais conceitos da linguagem, sempre
relacionando com o que foi visto em Programação I.

Cada pasta representa um tema específico e contém exemplos, explicações.
